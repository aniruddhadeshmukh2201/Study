# Python Basics for Practice

Python is a high-level programming language that is popular for automation, data analysis, web development, and machine learning.

When you work on a Python project, it is best to create a virtual environment. A virtual environment keeps project dependencies isolated from other projects on the same machine.

Useful commands:

- Create a virtual environment with `python -m venv .venv`
- Activate it on Windows PowerShell with `.venv\Scripts\Activate.ps1`
- Install packages with `pip install package-name`

Python files usually end with the `.py` extension.
