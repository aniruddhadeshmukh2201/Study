# GenAI Study Notes

GenAI workflows usually involve prompt design, retrieval, grounding, and evaluation.

Chunking is important because large documents must be split into smaller pieces before retrieval.

Embeddings help compare text by meaning, but a lighter example can start with TF-IDF to understand the pipeline.

For a real GenAI app, you can later replace the simple synthesis step with an API call to an LLM provider.
